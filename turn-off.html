Turning Off JupyterHub and Resources¶

  1. If you want to stop these resources from running, you’ll need to tell Google Cloud to explicitly turn off the cluster that we have created. This is possible from the web console. Click on the hamburger menu icon (the 3 horizontal lines) in the top left, and then click on the Container Engine section (see figure). Select the container you wish to delete and press the “delete” button.

    Note

    Alternatively, you can run the following command to delete the cluster of your choice.

    gcloud container clusters delete YOUR_CLUSTER --zone=YOUR_ZONE

  2. Now your cluster resources should be gone after a few moments - double check this or you will continue to incur charges!
